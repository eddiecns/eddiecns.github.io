002
Home Lab — Full DevOps Pipeline on Proxmox ↗
Featured

7-stage CI/CD pipeline built from bare metal on a repurposed HP desktop — no cloud, no managed services. Terraform (Telmate) provisions VMs from a Rocky Linux 9 cloud-init template on Proxmox. Ansible hardens SSH and configures every node. GitLab webhooks to Jenkins for CI: ephemeral MySQL, migrations, tests, docker save to .tar, distributed via Ansible LXC to a 4-node kubeadm cluster. Prometheus + Grafana monitor everything.
