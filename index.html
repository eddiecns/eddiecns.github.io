002

Home Lab - Full DevOps Pipeline on Proxmox

A complete 7-stage CI/CD pipeline built from bare metal at home - no cloud account, no managed services. Terraform provisions all VMs by cloning a custom Rocky Linux 9 golden template on Proxmox. Ansible hardens SSH and configures every node. Code is pushed to a self-hosted GitLab, which webhooks to Jenkins for CI: ephemeral MySQL spins up, migrations run, unit tests execute, and a Docker image is exported as a .tar. Ansible then distributes the image and Kubernetes configs from the LXC control node to the K8s master, which orchestrates a rolling deployment across 4 nodes - MySQL on worker-3, Django + Gunicorn + Nginx on the rest. Prometheus scrapes node-exporter and kube-state-metrics; Grafana visualises everything on a dedicated monitoring VM. Every commit triggers the full pipeline automatically.

005

Student Grade Calculator - Android App

An Android digital gradebook for teachers, built in Android Studio with Java. Teachers enter student details and subjects with marks and credit hours - the app automatically converts marks to grade points, calculates GPA using the weighted formula (∑grade points × credit hours / total credit hours), and classifies students into grade categories A through F. All data is persisted in a structured SQLite database with proper relational tables between students and subjects. Includes search functionality by name or ID, AlertDialog-based views, and class-level statistics: average GPA, highest/lowest GPA, and grade distribution.

006

Business Inventory Management - Android App

A comprehensive Android inventory and sales management system demonstrating advanced mobile development. Features four main screens - Dashboard, Inventory, Sales, Reports - connected via bottom navigation. The SQLite database handles full CRUD on Products and Sales tables with automated real-time stock updates on every sale transaction and low-stock alert notifications. UI components include RecyclerView with custom adapters, input forms with validation, DatePicker, confirmation dialogs, and a professional splash screen. Generates daily/weekly sales reports with profit margins, inventory valuations, data visualisation, and CSV export. Consistent theming and custom branding throughout.
